Testing HTML5 Semantic tags


The main tag

For the main content of a webpage, unique to that page

The section tag

Defines a certain section of a webpage (ex: blog list, contat info)

The article tag

Defines a bit of content which makes up an article (ex: a blog post)

The aside tag

Defines some content related to something else (ex: similar blogs)

The header tag

For the header of a website, contains the nav, title etc.

The footer tag

For the footer of a website.
